# git 初阶使用

## 快速掌握

bash的复制粘贴

```bash
ctrl + insert  # 复制
shift + insert  # 粘贴
```



```bash
git clone ssh链接   ## 注意会创建一个与repo同名的文件夹

git status ## 查看本地与远端仓库（最新）之间的状态差异

git add . ## 将所有更改（除了gitigonre）增加到暂存区

git commit -m "本次提交的介绍" # 提交

git push  # 提交远程仓库


```
